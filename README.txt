xulate
================================================================

This program accepts a XuLA UCF file via the standard input and
converts the pin assignments to ones appropriate for the XuLA2.
The modified UCF is sent to the standard output:

	xulate < XuLA.ucf > XuLA2.ucf

You can also use arguments for the input and output UCF files:

	xulate -i XuLA.ucf -o Xula.ucf
